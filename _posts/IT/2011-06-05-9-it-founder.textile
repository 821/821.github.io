如果你沒聽說過北大方正，說明你很可能不是中國人，至少不是一個留心國內社會的中國人。
但如果你瞭解北大方正的產品，說明你絕對不是 99.9% 的中國人，因為一般的中國人對他實在沒什麼印象。
不過如果你完全沒有接觸過北大方正產品做出來的東西，那你絕對不是中國人，甚至不會是華人，至少你不可能認得中文。
為什麼我能夠非常確定的說出第三句話呢？
因為中國所有的出版社都在使用方正書版，而臺港也有不少出版社使用了方正飛騰，現行很多比較好看的中文字體都是方正的作品（有人說，如果使用版權全都要收費，方正可以開著大卡車在全國所有打印店一家一家收保護費了）。如果你完全沒接觸，你要怎麼讀書呢？別說讀書了，看一下廣告都不小心見到方正的。
知道方正的厲害了吧。

既然說方正，那就提一下王選吧。王選是何許人也？「漢字激光排照系統的創始人。」很多中國人一定會搶答。還有呢？不知道了吧？而且我相信，很多中國人對「漢字激光排照系統」沒什麼概念，「反正就是漢字和電腦神馬的吧」。
我不專業的隨口說一下好了。相信 80 後一定記得，很久以前，每次學校段考的時候，那個試卷都很噁心，手一碰，就掉顏色，而且把手弄得很髒。這個東西叫做鉛印，是老技術。而我們現在看的書，以及平時打印的，都是激光排印的。
如果沒有王選，你電腦上想裝個能打中文的打印機都不行。王選的外號叫「現代畢昇」，畢你個頭昇，畢昇的活字印刷根本不普及，能跟王選比嗎？畢昇纔是「古代王選」哦。
一句話：在中國，影響最廣的軟件公司，不是騰訊，不是百度，是方正這個大家衹知道個名字的公司。

牛皮吹完，下面就說說我瞭解方正的過程吧。
我有一個特質，那就是，在東西的外觀上，我的喜好和主流非常不同，穿衣戴帽、軟件介面、文字排版，全都不同。也許你覺得沒什麼大不了。確實，平時真的沒什麼大不了的，最多被人「另眼相看」而已。可是一旦做起排版來，就麻煩了：我找不到合適的模版，甚至曾經找不到合適的軟件，來排版我的小說。
從前讀中華書局改革開放初期的古文觀止，就覺得雙行夾注是注文不多時注釋的最佳模式，因為讀者可以很輕易的讀到注文，而不必反復翻動書葉。這種模式在錢穆的國史大綱中發揮得最好，他的注釋不多，不會使正文看起來太過稀疏，總之非常賞心悅目就是了。另外看脂批紅樓夢時，眉批給我的印象很深，一直熱愛。最後廢話一下，我的小說當然是直排的了（不然眉批個屁啊），順便頁碼也給個中文的吧，還有標點也居中一下。
很簡單是不是。
TMD 一點都不簡單。你用 Word 排給我看看啊？還是用 TeX ？再或者 Adobe Indesign ？
我可以很確定的告訴你，以上三種軟件雖然代表了三個行業巨頭，但很不幸，我都沒有採納。
首先是 Word 。易用性高，凡有電腦的地方，必然可以打開 Word 生成的 doc 文件。並且 Word 的中文支持還是不錯的，至少轉直排很容易。 Word 的功能也很多，不過來來去去都是英文世界比較喜歡的功能，總之他有的我基本不用，我用的他基本沒有就是了。人家不是專業排版軟件，不能強求。
看完很方便但有點弱智的 Word ，我們來看看偉大的 TeX 。原始作者高德納是厲害的人， TeX 語言也是非常偉大的語言（不瞭解這門語言的人可以通過 html 來理解，如果這都不懂，那我也沒辦法了，總之就是在一段文字前後加上一些命令之後，排版出來的樣子會不同，從而達到排版的目的）。不過我想說的是， TeX 並不好學，而且高德納是個人而非公司，他的版本沒法用，用戶都使用改進版，版本非常複雜，新手很難找全所有需要的宏包，就算找全了，不常折騰軟件的人恐怕還不會裝。另外， TeX 的中文支持非常噁心，直排都很難正確實現 [1] 。如果你真的用過 TeX 來排中文，特別是寫自己的東西，一定有過想掀桌砸電腦的情緒，因為他排出來的東西總是和你想像的有那麼一點半點乃至十點八點差距，甚至你一排他就跟你說你搞錯了，不讓排，然後又不告訴你錯誤具體在哪裡，讓你自己猜（有時是宏包沒裝，有時是命令搞錯，有時字體出問題，不一而足，而錯誤提示往往不得要領，答非所問，需要 Google 纔知道他到底在說什麼）。另外， TeX 一大堆宏包裝下來，體積也不小了，然後你還得裝個設置好了的文本編輯器，不然他辣塊媽媽不開花，代碼死都不高亮。最後最後，就算你英明神武，不但會用 TeX ，而且還讀通了 TeXBook+source2e ，會搞開發，你還是會遇到一個問題：編譯速度。我不知道是宏包裝太多還是系統不好，總之排稍有古籍特色的東西，基本上是幾秒一頁的速度。老大啊，我排的東西動輒上百頁哦，而且修改期間會經常排來看看，你讓我怎麼活。「 TeX 不是一個用來製造比較漂亮的書籍的程序，而是用來製造最美的書籍的程序。」去你媽的最美，我要的是我的 style 。而且我是老人家，不是數學家。總之， TeX 排中文很痛苦，除非用到相當難入門又基本停滯的 ConTeXt MkII ，纔能解決大多數問題。
看完很厲害但有點殘廢的 TeX 系列，再看看 Adobe Indesign 好了。說老實話，磚頭公司的產品雖然渲染做得很好，卻讓我不大敢用，因為速度太差了。 Indesign 在西方報刊及廣告業的地位是不可否認的強大，不過我就發現了一個令我不想使用的原因：不方便修改。你看那殘廢的 TeX 啊，他多方便，你隨便那一個文本編輯器就可以修改了，而且存放起來幾方便。這對於經常反復修改的本人來說非常重要。可是 Indesign 他好死不死是所見即所得的，加上軟件本身速度問題，實在很嘔血啊。夾注還好，你讓我的眉批怎麼辦？用你我乾脆用 PowerPoint 了，反正都裝了……
總而言之，這些洋玩意是不行了。
一方面是好使但殘廢的 TeX 衍生物，一方面是高大全但不好使的 Indesign ，看完這兩個軟件，你大概會想，有沒有一款軟件，使用標記語言，而又適合排版書籍如 Indesign ？
答案是方正書版。
方正書版就是一個很可能基於 TeX 源代碼，但命令完全不同，專門為中國出版界設計的排版軟件。早在 DOS 時代，方正書版就已經出現，並且實現高度完善。實際上，方正書版從 1999 年的第九代到現在，他的 BD 語言變化非常小，主要是增加了一些細微的小功能，並在介面和格式轉換上進行少量修正，可見當年就已經相當完善了。
BD 語言如果從當下流行的標記語言來要求，確實不大規範。但是不規範也有不規範的優點，那就是很方便用。熟悉 html 的都知道，很多符號是需要代碼的，記代碼是很鬧心的。 BD 語言就不用，直接軟鍵盤一打，或者快捷鍵一敲，就可以了。常用的符號出現在工具列。
而且書版的易用性遠高於 TeX 。安裝宏包我就不說了， TeX 光是設定就夠你頭暈的。各種教程看遍，有沒有結果還是一回事呢。而書版就非常漂亮的解決了這個問題：他把書籍分成大樣和小樣，大樣利用一個 pro 文件來控制，小樣用 fbd 文件完成。 pro 文件可以用書版的編輯器進行選項式的編輯，不需要敲命令。你衹要在 pro 上點點鼠標，就能設定八種不同的標題模式，實在比 TeX 舒服太多了， TeX 要裝宏包然後自己想命令，而非原生支持。小樣就是書籍正文，用語言標注一下就排好了。如果你覺得這些描述性的語言不足以形容書版的好用，那我就用數字說明好了：我斷斷續續學了超過 400 天 TeX ，還是排不出稍微有點像我需要的樣子。而書版呢？一天。當然，你可以說，因為我有 TeX 的底子，所以書版學得很快。可是我學 TeX 的時候還有 html 的底子哦，怎麼不見我學得快？
用 TeX 的人大多苦於找不到好用的編輯器。在我看來，支持實時顯示的 Bakoma 是最好的 TeX 編輯器，不過他和書版一樣，都是收費的（而且對引擎有限制）。而書版自帶的編輯器還是挺好用的（雖然我衹在手冊上看過，畢竟平時都用 EmEditor ）：代碼高亮、正則表達式替換、豐富且可以自定義的快捷鍵、語言糾錯功能、非常高的開啟和編譯速度。
書版的編譯速度也非常可喜。我的電腦是 AMD 的低端雙核，內存 2G ，編譯一份一百多頁的中文文檔大概衹要一秒，基本上一點就出來了。不知為何，我編譯 TeX 總是不能這麼快，一秒鐘纔出一頁。可能 TeX 太自由，宏包太複雜，所以他自己暈倒了吧。
另外，如果你原本用 Word ，心血來潮決定用 TeX ，就要找專門的軟件來轉換，然後效果總是很不理想。而在書版中衹要另存為他兼容的 Word 版本就可以了。
這麼好用的書版，你認為他的體積有多大呢？不到一張光碟。而這張光碟之所以這麼大，是因為裝了字體。如果去掉字體，可以得到一個 36MB 的壓縮包。這是書版 10 。去掉不常用的 CEBMaker （一個沒有完美破解的轉換為方正獨有的 ceb 格式的雞肋軟件），衹有 5MB ，再去掉我用不上的模版和範例，好傢伙，都不到 5MB 了！我考，比很多文本編輯器還小（我用的壓縮軟件是 WinRAR ）。我一般就用兩三種字體，加起來也就 10MB 左右，一封郵件都解決了。而我用過一個九代的增強版，加了很多模版、範例的，也不到 14MB 。你知道 TeX 多大嗎？功能嚴重殘廢（中文都不能排）的 MiKTeX 最小安裝版都破 50MB ，真想知道這 50MB 是不是都安到豬身上了。一個完整版 CTeX 居然超過了 1GB ，這恐怕豬都覺得無法解釋了……（可惜我老人家不會畫畫，不然應該配一幅漫畫，畫的是用戶厲聲喝問豬把 TeX 藏哪裡去了，豬很無辜的說你就算把我喫了我也不知道。）
我初用書版的時候，曾經一天發了五條 Facebook 近況，盛讚書版，奚落其他。我這麼挑剔乃至龜毛的人都能發軟文，可見書版的厲害。而且九代是十二年前的老版本。這十二年間，不管是什麼軟件，都可以有改頭換面的進步，但九代書版依舊能在出版界大放異彩。

下面開始罵方正。
書版千好萬好，一點不好：極度不開放。
使用書版編譯之後，會得到一個 ps 文件，但是這個文件是加密的，衹有方正自己的軟件可以打開。在 2008 版中，又增加了轉為 pdf 的功能，但是這個 pdf 的每頁第一個字是圖片，所以屬於不完全開放的 pdf 。另外，我們在書版中雖然可以很哈皮的看到大樣，但是打印功能卻受到限制，需要使用破解補丁。
最神奇的是，方正有一款叫暢流的軟件可以剝離出一個 PDFCreator ，歷來是大家轉 pdf 的利器。但這個利器居然無法利用在安裝書版時已經裝好的方正字體，而需要專門裝一個 CID 並重置字體。方正的 PSPPro 也是這樣。我勒個擦的， PSPPro 是幾千大洋的高價產品，居然殘廢如此，我真的無語了。
這樣就限制了軟件和其他軟件互動的可能，使得方正的軟件成為一個極端封閉的體系，令不瞭解他的用戶望而卻步。當然，你也可以說這是方正用來推銷他的印前軟件的一種方式，什麼暢流、 PSPPro 、 PSPNT 之類，如果沒那麼多限制，就不太有市場了。
而方正的反盜版工作從來都是世界級的，採用安全狗方式反盜版。安全狗是個類似工行 U 盾的東西，必須把他插在電腦上，你的軟件纔可以使用。新版的安全狗使用最新的加密技術，目前就算是網上賣的盜版，也有一個完全複製了正版狗的盜版狗，使得盜版的成本都超過一百塊。
方正的這套強大系統，跟 Indesign 價格差不多，但是 Indesign 非常容易破解，網上到處都是，雖然比較菜，用戶卻非常多。用戶量一上去，銷售量上去也是遲早的事情。於是方正就這樣莫名其妙的被擊敗了：現在招排版員，會不會書版倒是其次， Indesign 倒是一定要會的。而且 Indesign 的破解版滿天飛，很多非專業人士也會裝一個，但書版雖然有舊版的破解版，卻僅在一些論壇的 FTP 能夠找到（而且門檻還挺高的）。
書版如此，飛騰、飛翔等其他產品亦然，不必一一贅述。
總而言之，諸多傻叉無聊又變態的設計，讓我感到方正公司非常之狹隘。雖然有一流的程序員，但公司的管理層目光短淺，個性極端封閉，根本就是古代「祖傳祕方」式管理，以及閉關鎖國的天朝心態。

另外，方正又是一個極度缺乏進取心的公司。我用過後認為，方正書版雖是 TeX 的本土衍生物，但絕對不僅僅適合出版社使用，也非常適合作家使用，是一款完全可以替代 Word 的軟件。其實衹要在書版內建一個非常簡單的替換功能，書版就可以導出 TeX 文章源碼（架構如此接近，當然很好轉換了）。加強一下 pro 文件設置，加上幻燈片，搞點 fancy 的圖片背景和字體、顏色樣式，用戶體驗估計得破表。最後把 pdf 轉換變成開放形式的，弄幾個皮膚，就可以去搶西方市場了。如果時機合適、廣告給力，搶掉 Word + Powerpoint + 收費 TeX + Indesign 半壁江山不見得是什麼很奇跡的事情。可以部分學習 M$ ，推出三種模式：衹能預覽的免費版，可以導出的用戶版（售價和 Office 家庭版差不多即可，大概 100 美金罷），集成了 PSPPro 或暢流等等並具有拼版之類一大堆我不知道是什麼功能的企業版（售價 1000 美金以上）。

一個強大的排版系統，雖然默默製作了無數的作品，卻終於因為消極、封閉、太出力打擊盜版，而終於影響了前途和錢途，這難道不是一種悲哀嗎？
