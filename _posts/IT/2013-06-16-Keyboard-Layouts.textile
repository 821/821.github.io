h2. Part 1. 鍵盤的佈局

聲明：如將本人自創佈局用於商業鍵盤，必須每樣送兩把作爲版權費。

鍵盤史上出過很多佈局，有奇葩的，有專業的，各種各樣，百花齊放。
但是現在廣受認可的無非幾種：
104/106/102: 104 是最標準的佈局， 106 是在空格左右兩側各加一個按鍵，構成日韓式鍵盤，而 102 則是省掉兩個 Win 鍵，多見於蘋果機。
104 又可以細分爲歐式和美式。美式是最標準的，不提，日韓也是美式基礎上的。歐式一般是特大回車，小退格，小反斜槓，小左 Shift ， Z 鍵左邊還多一個符號——對於用慣美式佈局的人來說，退格、左 Shift 、反斜槓這幾個鍵都是非常容易誤觸的。
87: 104 去掉數字小鍵盤。
84: 筆記本，具體到不同的機器上有不同的體現。

有鑑於本人和一般人共性不多，個性不少，對於經典的三種佈局當然是不滿意的。
首先， Ctrl 放角位， Ctrl+X, Ctrl+V, Ctrl+Z, Ctrl+F, Ctrl+H 極端不便，不夠重要的 Win 和 Alt 反而居於有利地位。
其次，經常用 Home End ，而他們在編輯區中間，易誤觸。
第三，方括號、大括號、引號幾乎不用，放主區太浪費。
第四， Esc 太遠。
第五，很多鍵幾乎沒用過。還有像 104 的 Num Lock ，按下去就讓燈一直亮着，沒熄過。
第六，一般的 104 佈局不帶多媒體功能，而關機、睡眠、音量這些還是比較有用的。
第七， 104 佈局臃腫，其他又沒有我偶爾會用的數字小鍵盤——我需要數字和方向同時可用。
第八，非常不滿於現有 ;: 和 /? 的關係，我不是程序猿，應該倒過來。
第九，鍵盤的最初設計者腦子絕對有問題， = 和 + 弄反了，結果現在輸入法都跟着反了。你還不能用 + 翻頁。
第十，空格鍵太長，易卡鍵。

本着我老人家善於想像，疏忽實踐的精神，當然要爲自己設計若干鍵盤佈局了。
圖例：
小框是燈。
紅色的應當側刻，在按下 Fn 的同時按纔出鍵。
藍色是按上檔，沒什麼好說。
綠色是兼容紅色的基礎上在 Num Lock 燈亮時取代主要鍵。
同時，空格鍵弄得很短，這樣手感會比較好。佈局保持了回格、回車、頓號是大鍵，不縮水，這樣不易誤擊。 Esc 的位置縮向主鍵區，使用 Vim 或類似軟件時會比較舒適。
Pn 是爲編程鍵盤留下的編程鍵

一開始就太霸氣不好，先設計一款大衆化的—— Slim Full 和 Power Mini 。
前者是 104 的變體，改動不大，但多媒體豐富。後者是比 87/84 更省略的一種設計。
槽點不多，不解釋了。
!../Pic/Normal.svg!

不過我沒必要照顧右手控，不是麼。 F1-F12 也不常用，不妨一同去掉。
於是 Left Comfort 誕生，標點符號也做了調整。小指直接 End ，右手大拇指也找到了工作。
而且經研究發現，大括號和方括號一般都不必輸入右側的——文本編輯器會自動輸入，所以放到 Fn 。
!../Pic/Comfort.svg!

但是有沒有覺得鍵盤普遍太扁長，不夠美觀呢？
而且 F 區也不是完全沒用，至少放在家裏而非扛出門的鍵盤，還是可以有。
所以我設計了一個 Writting Silver 。退格和回車變得非常好按，而核心區按鍵和標準鍵盤基本一樣。右手大拇指也分配了適當的工作。加號問題完美解決——既考慮了輸入法，也考慮了位置，還考慮了數字小鍵盤的方便性。
爲什麼叫 Silver 呢，因爲無邊框時長寬比是 2.7586 ，加上邊框這個數字會再小一點，而數學上的白銀分割是 2.4142 ，完全可用邊框實現。
!../Pic/Silver.svg!

鑑於 F 區確實用得少，乾脆衹保留最常用的 F5 好了。記得裝個鈦合金的~
回車大小恢復，但是繼續放在小拇指下。小鍵盤的回車是不是也要個大的呢？沒問題。
這就是 Pico ！
!../Pic/Pico.svg!

弱小的人類阻力在我老人家眼裏浮雲都不如，所以，發散思維繼續。
Silver Function 就是一個發散思維的產物。我非常討厭組合鍵，但是有的組合鍵確實經常用，乾脆定義幾個好了。
另外我設計了兩個奇葩的鍵，就是寫着 Mono 那兩個，比如按了一下 Ctrl Mono ，再按 F ，這時就相當於按了 Ctrl F 。啊哈，是不是有解放了的感覺？其實 Ctrl Mono 放進之前的設計都是可以的，畢竟前面都至少有兩個 Ctrl ，不過因爲之前沒有定義新鍵，所以沒加入。
另外輸入法翻頁明顯是 = 用得多，所以把他放到更舒適的位置。
當然，我這些組合鍵僅僅是提供一個發散的模式，一般的可編程按鍵也不錯。
!../Pic/Function.svg!

有了 Silver 就有 Gold 。再省掉一列試試看。於是就出現了 Writting Gold 。
由於過度精簡且保持數字區，整個鍵盤的設計難度就加大了。不過這下回車爽了，直接在小指之下。退格改成 7 字形，一個很容易擊中的形狀。等號的位置和標準鍵盤幾乎是一樣的，所以對於輸入法沒影響。
我比較不滿意的是，原來問號的地方放了上箭頭這個我不是特別常用的按鍵，如果是下箭頭就好了，可是倒過來又太沒邏輯。
其實指法正確的人使用這個佈局不會有太大的壓力，就是要適應一下。
由於回車變小，爲了恢復那種很爽的擊鍵感，在右下角又放回一個一字回車。當然這就把 0 擠到上方去了。
!../Pic/Gold.svg!

現在很流行人體工程學。鍵盤的人體工程學主要有幾個考量：一是使用普通鍵盤時，手臂是彎曲的，手腕也不直，所以久了會手腕痛。二是普通的 104 鍵盤太肥，如果左手用鼠標還好，右手的話，還隔着編輯區及數字區，右手活動比較多。三是手指長短不一，而同一排的鍵位卻是平的，手指需要非常彎曲，纔能保證敲擊準確，這當然也對手指有傷害。
針對第一種情況，現在的做法是拉開左右手距離——要麼是把很多鍵放中間，要麼是鍵盤可分離。第二是去掉數字小鍵盤，但是這樣顯然是不夠方便的。第三是讓同一排的按鍵有所起伏，中指的就遠一點，小指的就近一點。
我看到分離式，第一反應就是，何不再繼續分下去？鍵盤完全可以做成三個部分，最好做成藍牙、無線那類，任意活動。
分離後，發現其實又多出來一些很好的鍵位——食指內側。這些鍵位拿來打字肯定不行，放次常用鍵卻非常好。這個佈局配合我定義的按鍵很好的體現了作者情懷。
!../Pic/Divided.svg!


要繼續發散當然不難，不過發散已經到了一個高峯，再發散下去就太神奇了，還是暫時打住罷。

h3. 薄膜

Dell SK-8115: USB, 廣受好評的便宜貨，停產，幾乎全是假貨。同型號的中國版爲羣光代工，差很多。
Dell QuietKey RT7D5JTW: 偏硬，清脆，停產，有庫存。
IBM SK-8815: USB, 不硬，略肉，帶手托和兩個 USB 口，停產，有庫存。
IBM SK-8820: 偏硬，停產，有庫存。
IBM SK-8806: USB, 偏硬，帶手托和兩個 USB 口，停產，有庫存。不是標準的 104 鍵位。
IBM KB-7353, KB-7953, KB-9910: PS2, 偏軟，帶手托。這個系列衹有最後一款有庫存。
IBM KB-6323, KB-8923, KB-3923: PS2, 最經典的一個系列，用料一個比一個縮水，最後一個有庫存。
Lenovo SK-8820, KU-0225: 前者 PS2, 後者 USB, 前者偏硬，後者軟，有手托。不過個人對 KU-0225 的評價是，普通使用兩年就很澀。
明基海灣： USB, 剪刀腳架構，鍵程比筆記本的長，有點肉，停產。與之相似的還有海貝，但是海貝的形狀不好，難按。
Compaq SK-2800, SK-2700: PS2, 舒適柔軟的長鍵程鍵盤，停產。
Compaq RT6656: 廠家是薄膜鍵盤玩家相當推崇的 NMB ，綠色膠碗，回彈給力。
Compaq KU-9978: USB, 停產，有庫存，多媒體。
Compaq KB-3923: PS2, 類似 IBM KB-3923
HP SK-0316, KU-0316: 偏軟，有鋼板，停產，有庫存，假貨較多。
HP SK-2880: PS2, 偏硬，有鋼板，停產，有庫存。
HP SK-2502: PS2, 停產，這貨有類似機械的成分，所以質量好，耐用。
HP KB-5187: PS2, 偏軟，停產，有庫存。
HP C1405A: PS2, 停產，僅舊貨。產於新加坡，熱昇華鍵帽。
SUN TYPE 6: 八針圓口或 USB, 特軟，回彈差，鍵程短，熱昇華鍵帽， Unix 佈局，部分按鍵在 Windows 無效。停產，有庫存。
Key Tronic: PS2, 分區壓力，空格鍵那一行很重口味，特別是中國版，部分還有庫存。著名型號有 LTC 和 LTD 等。
Microsoft 2000, Microsoft 3000, Microsoft 4000, Microsoft NKE, Microsoft NKP: 軟，人體工學。性價比差點。
DT-35 三星韓版：較硬，回彈好，現在的不行了。
IBM Model M: PS2 聲音很大，特硬。雙層熱昇華鍵帽，機械薄膜，又稱屈蹲彈簧軸。分爲四個時代：第一是金屬標，在右上角有個金屬標，生產日期爲 1985-1986 年。第二是灰標，由 IBM 自己做。第三是藍標， Lexmark 代工。第四是現代，專利轉讓了，廠商是 Unicomp 。灰標基本上僅有洋垃圾，藍標少量新品，第三版還在生產。
IBM M13: 基本上就是 Model M ，但是多了 IBM 特有的貓舌。
AT&T 305B: Model M 的定製版，據說比較輕。
Fujitsu FKB4725, FKB4726: Model M 的復刻版，其實不像的，彈簧短，而且有膠在裏面。工藝上 4725 是熱昇華， 4726是激光刻字，但材料都是 PBT 。由於熱昇華現在比較貴，所以 4725 貴 100 左右。均已停產，但還有全新品，而且比藍標 IBM Model M 便宜。現在售價仍然低於當年他還在生產時。
Fujitsu Libertouch: 在售。可以自行調整每個按鍵壓力的薄膜鍵盤。每個按鍵下面有一個膠碗和彈簧，並且可以自行更換。鍵帽爲 PBT ，激光刻字。不過價格麼……

h3. 機械

Cherry 的鍵盤沒有差的，但仍然以無鋼板、無段落感、輕型軸、 PBT 爲最具特色。中端的 3000 系列，低端的 3800 系列，高端的其他系列，都有非常好的表現。其中 3494 的 PBT 版尤被玩家追捧，因爲他同時具備了四大元素，而且價格還在屌絲能夠接受的範圍內。
Filco 以鋼板有段落感鍵盤爲特色，其青軸和茶軸最佳。不過該廠價格虛高不下。
Cooler Master Quickfire Rapid 87 和 Quickfire XT 104 兩個型號與 Filco 的類似款不但同廠，而且內臟幾乎一樣，但中國定價便宜不少。
DAS 四代名聲非常好，青軸尤其著名，茶軸也不錯。
Razer Black Widow 87 使用改良版青軸，彈簧壓力較小，而且無背光版的價位低，非常具有性價比。
Plu 87 的青軸和茶軸是性價比較高的國產機械，雖然用料不佳，但是手感還不錯。
HPE 87 內臟用料較好，國產良心。不過已經停產。
Kinesis Advantage 是茶軸人體工學鍵盤，很貴。
Truly Ergonomic 也是人體工學鍵盤，還是貴。
